# Entropy

## A 96% Southpaw keyboard, with split space and encoder support. 

Status: Tested and working. 3D printed enclosure WIP.

## Features:
- Encoder support in 0,0 -> 0,3 + 0,19
- STM32F303 MCU with QMK and VIA Support (not upstream yet)
